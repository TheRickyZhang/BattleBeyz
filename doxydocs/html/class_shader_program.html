◆ setObjectColor()

void ShaderProgram::setObjectColor ( const glm::vec3 * color) const

Set object color.

This sets the useObjectColor flag and objectColor vectors in the shader program object.vs. This will override the colors defined in the vertex data.

IMPORTANT: Turns this off when you are finished using it!

Parameters
color[in] Pointer to color. If null, the objectColor handing is disabled.
